========================================
WEEKLY WORKOUT ROUTINE FEATURE
========================================

Create a smart weekly workout planning system.

Features:
- Trainers can assign weekly workout routines
- Members can view weekly schedules
- Calendar-based workout planner
- Rest day planning
- Muscle group split planning
- Push reminders for workout days
- Workout completion tracking
- Weekly progress analytics
- Auto progression tracking
- AI-generated weekly workout routines

Workout routine examples:
- Push Pull Legs
- Bro Split
- Upper Lower
- Fat Loss Routine
- Muscle Gain Routine
- Beginner Routine
- Home Workout Routine

Each weekly plan should include:
- Exercise name
- Sets
- Reps
- Rest time
- Exercise videos
- Notes from trainer
- Workout duration
- Calories burned estimate

INBODY REPORT ANALYSIS FEATURE
========================================

Create a feature where users can upload their InBody/body composition report.

Supported uploads:
- PDF
- Image
- Camera scan

The system should:
- Extract body metrics using AI/OCR
- Analyze:
  - Weight
  - Skeletal muscle mass
  - Body fat percentage
  - BMI
  - Visceral fat
  - Water retention
  - Protein level
  - Metabolism
  - Segmental muscle analysis

After analysis, show a smart popup/modal:

"We analyzed your InBody report.
Would you like:
1. Personalized Trainer Guidance
2. AI Generated Fitness Plan"

Buttons:
- Connect with Trainer
- Use AI Plan
